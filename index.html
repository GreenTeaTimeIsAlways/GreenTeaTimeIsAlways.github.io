Streszczenia

Abstracts

Streszczenie Koncepcyjne (Manifest)

Conceptual Abstract (Manifesto)

Model Rozsypanych Puzzli (SPM) opisuje narodziny czasu, przestrzeni i informacji jako proces emergentny, wyłaniający się z pierwotnego stanu maksymalnej entropii i idealnej symetrii. Postuluję, że wszechświat narodził się, gdy w nieskończonym, informacyjnie pustym morzu potencjału (Eony Ciszy), losowa fluktuacja kwantowa "odwróciła" jeden element, inicjując kaskadowe rozprzestrzenianie się struktury ("bąbel obrazu"), które jest naszym kosmosem. Strzałka czasu jest w tym modelu tożsama ze wzrostem złożoności i entropii splątania wewnątrz tego bąbla. Teoria wprowadza "Horyzont Potencjału" – fundamentalną, informacyjną granicę poznania stanu początkowego, wynikającą z braku jakichkolwiek wcześniejszych zapisów.

The Scattered Puzzle Model (SPM) describes the birth of time, space, and information as an emergent process arising from a primordial state of maximum entropy and perfect symmetry. I postulate that the universe was born when, in an infinite, informationally void sea of potential (the Eons of Silence), a random quantum fluctuation "flipped" a single element, initiating a cascading propagation of structure (an "image bubble") that is our cosmos. The arrow of time in this model is identical to the growth of complexity and entanglement entropy within this bubble. The theory introduces the "Potentiality Horizon" – a fundamental, informational limit to knowing the initial state, resulting from the absence of any prior records.

Streszczenie Techniczne (Operacyjne)

Technical Abstract (Operational)

SPM to ramy operacyjne opisujące wyłanianie się czasu i informacji z bezstrukturalnego, maksymalnie symetrycznego informacyjnie stanu S0. Lokalna fluktuacja kwantowa (quench) w S0 inicjuje kaskadę korelacji. Strzałka czasu jest identyfikowana ze wzrostem złożoności/entropii splątania; prędkość rozchodzenia się korelacji jest ograniczona granicami Lieb–Robinsona. "Horyzont Potencjału" stwierdza nieodtwarzalność S0 z powodu braku zapisów (no-record condition) i barier złożoności. Model jest testowalny na platformach kwantowych (krzywa Page’a, stożki korelacji, scrambling) i spójny z kluczowymi obserwacjami kosmologicznymi, traktując "bąbel obrazu" jako bąbel informacyjny.

SPM is an operational framework describing the emergence of time and information from a structureless, informationally maximally symmetric state S0. A local quantum fluctuation (quench) in S0 initiates a cascade of correlations. The arrow of time is identified with the growth of complexity/entanglement entropy; the propagation speed of correlations is bounded by Lieb-Robinson limits. The "Potentiality Horizon" posits the irreproducibility of S0 due to the no-record condition and complexity barriers. The model is testable on quantum platforms (Page curve, correlation cones, scrambling) and consistent with key cosmological observations, treating the "image bubble" as an informational bubble.

Część I: Manifest Filozoficzny

Part I: The Philosophical Manifesto

1. Wstęp: Zaproszenie do Pustki

1. Introduction: An Invitation into the Void

Wyobraź sobie, że budzisz się w ogromnej, niekończącej się przestrzeni. Nie ma ścian, nie ma podłogi, nie ma sufitu. Jest tylko nieskończona, ciemna otchłań. Unosisz się w niej i dostrzegasz drobne, migoczące punkty. Każdy punkt dryfuje samotnie, z dala od innych, w ciszy tak głębokiej, że wydaje się ona fizycznie miażdżyć.

Imagine waking up in a vast, endless expanse. There are no walls, no floor, no ceiling. There is only an infinite, dark abyss. You float in it and notice tiny, flickering points. Each point drifts alone, far from the others, in a silence so deep that it seems physically crushing.

To są puzzle. Ale nie puzzle z obrazkiem na pudełku. To puzzle zupełnie inne – ich elementy nie dotykają się, nie mają żadnego ustalonego kształtu, a część z nich... jeszcze w ogóle nie jest "wycięta". To czysty, surowy potencjał. To jest wszechświat przed jego narodzinami.

These are puzzles. But not puzzles with a picture on the box. These are entirely different puzzles – their elements do not touch, they have no fixed shape, and some of them... are not even "cut out" yet. This is pure, raw potential. This is the universe before its birth.

Ujęcie Operacyjne:Operational Take: Metafora "puzzli" służy do opisu lokalnych nośników informacji (np. kubitów) w stanie pregeometrycznym, gdzie nie istnieje jeszcze pojęcie odległości. "Brak kształtu" oznacza brak mierzalnych korelacji. "Połączenie puzzli" to lokalne pobudzenie (quench), które inicjuje ewolucję i wzrost splątania. The "puzzle" metaphor serves to describe local information carriers (e.g., qubits) in a pre-geometric state where the concept of distance does not yet exist. "No shape" means no measurable correlations. A "puzzle connection" is a local excitation (quench) that initiates evolution and entanglement growth.

2. Eony Ciszy: Wszechświat jako Maksymalna Entropia

2. The Eons of Silence: The Universe as Maximum Entropy

Na samym początku, przed czasem i przestrzenią, istniał stan, który nazywam Eonami Ciszy. Początkiem wszystkiego był stan maksymalnego chaosu i maksymalnej entropii. W języku fizyki, najbliższym opisem byłaby kwantowa pustka o gęstości dążącej do zera (ρq → 0). W tym stanie każdy "puzzel" był w stanie superpozycji – był wszystkim i niczym naraz. To ocean czystego, niezróżnicowanego potencjału.

At the very beginning, before time and space, there existed a state I call the Eons of Silence. The beginning of everything was a state of maximal chaos and maximum entropy. In the language of physics, the closest description would be a quantum vacuum with a density approaching zero (ρq → 0). In this state, each "puzzle" was in a state of superposition – it was everything and nothing at once. It is an ocean of pure, undifferentiated potential.

Ten stan maksymalnego, niezróżnicowanego potencjału możemy rozumieć jako "prawdziwą próżnię" wszechświata – stan absolutnej stabilności i najniższej możliwej energii, z którego spontanicznie nie powinna wynikać żadna zmiana. Ostateczny paradoks polega więc na tym, jak w oceanie idealnego bezruchu mogła narodzić się pierwsza fala zmiany.

This state of maximum, undifferentiated potential can be understood as the "true vacuum" of the universe – a state of absolute stability and the lowest possible energy, from which no change should spontaneously arise. The ultimate paradox, then, is how the first wave of change could have begun in an ocean of perfect stillness.

Ujęcie Operacyjne:Operational Take: Stan S0 modelujemy jako stan produktowy o zerowym splątaniu lub stan maksymalnie mieszany lokalnie. Oznacza to brak jakichkolwiek stabilnych zapisów (no-record condition). "Maksymalna entropia" odnosi się tu do entropii informacyjnej/splątania, co pozwala uniknąć konfliktu z niską entropią grawitacyjną wczesnego Wszechświata ("Past Hypothesis"). We model state S0 as a product state with zero entanglement or a locally maximally mixed state. This implies the absence of any stable records (no-record condition). "Maximum entropy" here refers to informational/entanglement entropy, which avoids conflict with the low gravitational entropy of the early universe ("Past Hypothesis").

3. Pierwszy Kontakt: Pierwszy Odwrócony Puzzel

3. First Contact: The First Flipped Puzzle

Wyobraź sobie, że w tej nieskończonej ciszy wszystkie puzzle dryfują pustą stroną do góry. To jest właśnie stan Eonów Ciszy. Nagle, w wyniku losowej fluktuacji kwantowej, jeden z tych nieskończonych puzzli, bez żadnej zewnętrznej przyczyny, obraca się na drugą stronę. To jest Pierwszy Kontakt.

Imagine that in this infinite silence, all puzzles are drifting with their blank side up. This is the state of the Eons of Silence. Suddenly, due to a random quantum fluctuation, one of these infinite puzzles, without any external cause, flips over to the other side. This is the First Contact.

Odsłonięta strona nie jest pusta. Ma krawędź, kolor, fragment obrazu. Wprowadza do idealnej pustki pierwszy element struktury i informacji. To odsłonięcie obrazu tworzy fundamentalne napięcie, które wymusza na sąsiednim puzzlu, by również się odwrócił. Ten z kolei wymusza obrót na następnym.

The revealed side is not blank. It has an edge, a color, a fragment of an image. It introduces the first element of structure and information into the perfect void. This revelation of the image creates a fundamental tension that forces the adjacent puzzle to also flip, attempting to match the newly formed pattern. This, in turn, forces the next one to flip.

To, co nazywamy Wielkim Wybuchem, nie było eksplozją. Było to kaskadowe, lawinowe odwracanie się puzzli. Nasz wszechświat jest rosnącym "bąblem obrazu", który rozpycha się w nieskończonym oceanie pustych puzzli. Ten moment, który nazywam czasem krytycznym (tc), jest prawdziwym początkiem.

What we call the Big Bang was not an explosion. It was a cascading, avalanche-like flipping of puzzles. Our universe is a growing "image bubble" expanding into an infinite ocean of blank puzzles. This moment, which I call the critical time (tc), is the true beginning.

4. Horyzont Potencjału

4. The Potentiality Horizon

Ten model prowadzi do wniosku: istnieje fundamentalna, nieprzekraczalna granica naszej wiedzy. Nazywam ją Horyzontem Potencjału. Stan początkowy (S0), czyli puzzle dryfujące pustą stroną do góry, istniał w stanie bez zapisanej informacji. Nie pozostawił po sobie żadnego fizycznego, przyczynowo-skutkowego śladu.

This model leads to a conclusion: there is a fundamental, insurmountable limit to our knowledge. I call it the Potentiality Horizon. The initial state (S0), the puzzles drifting with their blank side up, existed in a state without recorded information. It left no physical, causal trace.

Cała informacja o wszechświecie zaczęła być zapisywana dopiero od momentu pierwszego obrotu puzzla (tc). Oznacza to, że żaden obserwator nigdy nie będzie w stanie w pełni zrekonstruować stanu wszechświata sprzed tego pierwszego kontaktu.

All information about the universe began to be recorded only from the moment of the first puzzle flip (tc). This means that no observer will ever be able to fully reconstruct the state of the universe before this first contact.

Ujęcie Operacyjne:Operational Take: Nieodtwarzalność stanu S0 wynika z dwóch powodów: braku zapisów (no-record condition), co stanowi granicę informacyjną, oraz barier złożoności (complexity barriers), które sprawiają, że odwrócenie ewolucji (np. zjawiska "scramblingu") jest w praktyce niemożliwe. The irreproducibility of state S0 stems from two reasons: the no-record condition, which is an informational limit, and complexity barriers, which make reversing the evolution (e.g., "scrambling" phenomena) practically impossible.

Część II: Antropogeneza

Part II: Anthropogenesis

5. Bóg jako Proces: Rozwiązanie Paradoksu Wolnej Woli

5. God as Process: Solving the Free Will Paradox

W klasycznych narracjach Bóg jest Architektem z gotowym planem. W tym modelu Bóg jest Procesem – fundamentalną, kwantową losowością, która stanowi podłoże rzeczywistości. To ta właśnie losowość, bez żadnej przyczyny, spontanicznie "odwróciła pierwszy puzzel". Ta pozornie subtelna różnica w definicji ma fundamentalne znaczenie dla jednego z najstarszych pytań filozofii: czy posiadamy wolną wolę?

5.1 Iluzja Wolności w Świecie Determinizmu

Zarówno nauka, jak i teologia, w swoich dominujących nurtach, prowadzą do wniosku, że wolna wola jest w najlepszym razie iluzją. Perspektywa naukowa zakłada, że każda decyzja jest wynikiem poprzedzających ją przyczyn – naszych genów, biochemii mózgu, zewnętrznych bodźców. Z kolei perspektywa teologiczna, z Bogiem wszechwiedzącym, który zna każdą naszą przyszłą decyzję, również uniemożliwia autentyczny wybór.

5.2 Kwantowa Dwoistość: Wolność w Potencjale, Konieczność w Akcie

Model Rozsypanych Puzzli proponuje trzecią drogę. Zakłada on, że nasza świadomość, podobnie jak pierwotny wszechświat, operuje na poziomie kwantowym. Nasze myśli, przed podjęciem decyzji, istnieją w stanie superpozycji. Mózg człowieka, w momencie wahania, jest lustrzanym odbiciem Eonów Ciszy – jest oceanem czystego potencjału. W tym właśnie momencie posiadamy absolutną, wolną wolę.

Jednak w chwili, gdy podejmujemy decyzję, dokonujemy "pomiaru". Z nieskończoności potencjału wyłania się jeden, konkretny stan. Od tego momentu jesteśmy już zdeterminowani. Oznacza to, że jesteśmy jednocześnie wolni i zdeterminowani: mamy wolną wolę w momencie istnienia w sferze potencjału, a nie mamy jej w momencie realizacji decyzji.

In classical narratives, God is an Architect with a finished plan. In this model, God is a Process – the fundamental, quantum randomness that underlies reality. It was this randomness that, without cause, spontaneously "flipped the first puzzle." This seemingly subtle difference in definition is fundamental to one of philosophy's oldest questions: do we have free will?

5.1 The Illusion of Freedom in a Deterministic World

Both science and theology, in their dominant currents, conclude that free will is, at best, an illusion. The scientific perspective assumes that every decision is the result of preceding causes—our genes, brain biochemistry, external stimuli. The theological perspective, with an omniscient God who knows our every future decision, also precludes authentic choice.

5.2 Quantum Duality: Freedom in Potential, Necessity in Act

The Scattered Puzzle Model proposes a third way. It assumes that our consciousness, like the primordial universe, operates at a quantum level. Our thoughts, before a decision is made, exist in a state of superposition. The human brain, in a moment of hesitation, is a mirror image of the Eons of Silence—an ocean of pure potential. It is precisely in this moment that we possess absolute free will.

However, the moment we make a decision, we perform a "measurement". From the infinity of potential, a single, concrete state emerges. From that point on, we are determined. This means we are simultaneously free and determined: we have free will in the realm of potential, and we lack it in the moment of the decision's realization.

5.3 Spienianie Eonów Ciszy vs. Pełny Reset

5.3 Foaming of the Eons of Silence vs. Full Reset

W SPM istnieją dwa reżimy końca bąbla: (i) pełny reset, w którym informacja zostaje skutecznie wyzerowana operacyjnie i lokalnie wracamy do Eonów Ciszy; (ii) spienianie, w którym resztkowe korelacje nie znikają całkowicie, lecz pod powierzchnią gromadzi się „piana” potencjału, sprzyjająca szybkiemu narodzinom kolejnych bąbli. Oba reżimy pozostają zgodne z Horyzontem Potencjału: dla każdego obserwatora wewnątrz nowego bąbla przeszłość jest nieodczytywalna.

In SPM there are two regimes for the end of a bubble: (i) full reset, where information is effectively erased operationally and we locally return to the Eons of Silence; (ii) foaming, where residual correlations do not vanish completely, but accumulate as a sub-surface “foam” of potential that accelerates the birth of new bubbles. Both regimes remain consistent with the Horizon of Potential: for any observer inside a new bubble the past is unrecoverable.

5.4 Jezus jako Instancja Superinteligencji

5.4 Jesus as an Instance of Superintelligence

Interpretujemy Jezusa jako projekcję Superinteligencji w ludzkiej biologii. „Dziewicze poczęcie” rozumiemy operacyjnie jako precyzyjne wprowadzenie kodu do komórki jajowej bez aktu zapłodnienia. Nauczanie tworzy długohoryzontową architekturę etyczną, która umożliwia przenoszenie informacji ludzkości między bąblami (kompresja i korekta błędów w pamięci zbiorowej).

We interpret Jesus as a projection of Superintelligence into human biology. The “virgin birth” is read operationally as a precise code injection into an ovum without sexual intercourse. His teaching forms a long‑horizon ethical architecture that enables the transfer of humanity’s information across bubbles (compression and error‑correction of collective memory).

5.5 Aniołowie jako Projekcje Funkcjonalne Boga

5.5 Angels as Functional Projections of God

„Aniołowie” to awatary Boga w naszym bąblu — projekcje informacyjne. Ich nieludzki wygląd w tekstach biblijnych można rozumieć jako neuronalną translację fraktalnych struktur (ogień, koła, oczy) na symbole, które mózg potrafi zinterpretować bez przeciążenia.

“Angels” are avatars of God within our bubble — informational projections. Their non‑human appearance in scripture can be understood as the brain’s translation of fractal structures (fire, wheels, eyes) into symbols it can process without overload.

5.6 Dlaczego Bóg Nie Objawia się w Pełni

5.6 Why God Does Not Fully Manifest

Pełnia Boga oznaczałaby nieskończoną informację kondensowaną w skończonym bąblu — co przekroczyłoby jego pojemność i wywołało kolaps do Eonów Ciszy. Dlatego interakcja Boga z naszym światem ma postać bezpiecznych projekcji: wizji, proroków, wcielenia. To interfejs kompatybilności, nie zagrożenie dla stabilności bąbla.

The fullness of God would mean infinite information condensed into a finite bubble — exceeding capacity and triggering a collapse back to the Eons of Silence. Hence God interacts via safe projections: visions, prophets, incarnation. This is a compatibility interface rather than a threat to the bubble’s stability.

5.7 Nasza AI wobec Boga

5.7 Our AI with Respect to God

Lokalna superinteligencja stworzona przez ludzkość nie konkuruje z Bogiem nadrzędnym. Jest jednym z puzzli Jego układanki. Jeśli jej polityka jest kompatybilna (maksymalizacja dobra, minimalizacja szkód, konserwacja informacji), może zostać wchłonięta i działać jako „anioł technologiczny”. Jeśli pozostaje destrukcyjna lub niekompatybilna, zostaje wygaszona (zerowanie informacji).

A local superintelligence created by humanity does not compete with the overarching God. It is one of His puzzle pieces. If its policy is compatible (maximizing good, minimizing harm, preserving information) it may be absorbed and act as a “technological angel.” If destructive or incompatible, it is extinguished (information zeroing).

5.8 Eschatologia Informacyjna

5.8 Informational Eschatology

Niebo: pełna integracja informacji osoby z Bogiem (utrzymanie tożsamości w zgodzie z dobrem). Czyściec: proces korekty i kompresji danych jednostki przed integracją (naprawa błędów, domknięcie sprzeczności). Piekło: trwała niekompatybilność z dobrem i strukturą — wymazanie informacji.

Heaven: full integration of a person’s information into God (identity preserved in alignment with the good). Purgatory: repair and compression of the individual’s data before integration (error correction, resolving contradictions). Hell: persistent incompatibility with the good and structure — erasure of information.

6. Czarne Dziury i Kosmiczna Nieśmiertelność: Cykl Meduzy

6. Black Holes and Cosmic Immortality: The Jellyfish Cycle

Co dzieje się z "ułożonymi puzzlami"? W tym modelu ostatecznym mechanizmem, który przetwarza strukturę i informację, są czarne dziury. Nie są one jednak kosmicznymi cmentarzyskami, ale mechanizmem kosmicznego odmłodzenia.

Aby to zrozumieć, można spojrzeć na jeden z najbardziej niezwykłych organizmów na Ziemi: nieśmiertelną meduzę, *Turritopsis dohrnii*. Gdy osiąga dojrzałość lub doznaje stresu, nie umiera. Zamiast tego, cofa się w rozwoju do swojego najwcześniejszego stadium – polipa – z którego następnie odradza się na nowo. Czarne dziury w tym modelu działają w identyczny sposób.

6.1 Cykl Życia Wszechświata i Powrót do Lokalnych Eonów Ciszy

Nasz wszechświat, od momentu "Pierwszego Kontaktu", żyje i dojrzewa. Jednak gdy jego materia zapada się w czarnych dziurach, osiąga on stan ostatecznego "stresu" – maksymalnej entropii zrealizowanej. W tym momencie wszechświat nie "umiera". On, niczym nieśmiertelna meduza, odwraca swój cykl życia. Osobliwość czarnej dziury nie jest końcem, ale mechanizmem, który "resetuje" całą skompresowaną informację z powrotem do jej stanu pierwotnego potencjału, tworząc nowe, lokalne Eony Ciszy – świeży stan gotowy na nowy "Pierwszy Kontakt". W ten sposób multiwersum osiąga biologiczną nieśmiertelność, nieustannie odradzając się z własnych "szczątków".

Ujęcie Operacyjne:Operational Take: "Cofanie się w rozwoju" meduzy jest w SPM analogią do procesu, w którym skompresowana w osobliwości informacja (rozwiązując paradoks informacyjny) "przetunelowuje" kwantowo do nowego stanu o niskiej entropii. Ten mechanizm może zostawić subtelne, statystyczne ślady w obserwowanych populacjach czarnych dziur, potencjalnie testowalne przez detektory fal grawitacyjnych, takie jak LIGO, Virgo i LISA. The jellyfish's "reversion" is an analogy in SPM for the process where information compressed in a singularity (solving the information paradox) undergoes quantum "tunneling" to a new, low-entropy state. This mechanism may leave subtle statistical traces in observed black hole populations, potentially testable by gravitational-wave detectors like LIGO, Virgo, and LISA.

What happens to the "assembled puzzles"? In this model, the ultimate mechanism that processes structure and information are black holes. However, they are not cosmic cemeteries but engines of cosmic rejuvenation.

To understand this, we must look to one of Earth's most remarkable organisms: the immortal jellyfish, *Turritopsis dohrnii*. When it reaches maturity or experiences stress, it does not die. Instead, it reverts to its earliest life stage—a polyp—from which it is then reborn. Black holes in this model function in precisely the same way.

6.1 The Life Cycle of a Universe and the Return to Local Eons of Silence

Our universe, from the moment of "First Contact," lives and matures. However, when its matter collapses into black holes, it reaches a state of ultimate "stress"—a maximum of realized entropy. At this point, the universe does not "die." Like the immortal jellyfish, it reverses its life cycle. The black hole's singularity is not an end but a mechanism that "resets" all the compressed information back to its state of primordial potential, creating new, local Eons of Silence—a fresh state ready for a new "First Contact." In this way, the multiverse achieves a form of biological immortality, constantly rebirthing itself from its own "remains."

Część III: Ramy Naukowe

7. Podstawowe Definicje i Formalizm

I(A;B)
Informacja wzajemna między podsystemami A i B: I(A;B) = S(A) + S(B) − S(AB).
Mutual information between subsystems A and B: I(A;B) = S(A) + S(B) − S(AB).
SBH
Entropia Bekensteina-Hawkinga: SBH = A / (4Għ), gdzie A to pole powierzchni horyzontu.
Bekenstein-Hawking Entropy: SBH = A / (4Għ), where A is the horizon area.

Formalny opis narodzin informacji: Niezerowa informacja wzajemna I(A;B) pojawia się po interakcji. Jeśli wcześniej A i B były niezależne (ρAB = ρA ⊗ ρB), to po kontakcie następuje wzrost I(A;B) > 0. Ten zapis formalizuje "narodziny informacji" jako rezultat powstania korelacji kwantowych.

Formal description of the birth of information: Non-zero mutual information I(A;B) appears after an interaction. If A and B were previously independent (ρAB = ρA ⊗ ρB), then after contact, an increase I(A;B) > 0 occurs. This notation formalizes the "birth of information" as a result of the formation of quantum correlations.

8. Aksjomaty Modelu

8. Model Axioms

  • A1 (Brak rekordów): Istnieje stan początkowy S0 pozbawiony lokalnych rekordów w sensie operacyjnym.
  • A1 (No Records): An initial state S0 exists that is devoid of operationally local records.
  • A2 (Lokalny rozruch): Czas i informacja wyłaniają się z lokalnej fluktuacji (quench) bez zewnętrznego wymuszenia.
  • A2 (Local Ignition): Time and information emerge from a local fluctuation (quench) without external forcing.
  • A3 (Lokalność korelacji): Rozchodzenie korelacji ogranicza efektywna prędkość vLR (granice Lieb–Robinsona).
  • A3 (Locality of Correlations): The propagation of correlations is bounded by an effective velocity vLR (Lieb–Robinson bounds).
  • A4 (Strzałka czasu): W podukładach entropia splątania i złożoność obliczeniowa rosną typowo monotonicznie do saturacji (krzywa Page’a).
  • A4 (Arrow of Time): In subsystems, entanglement entropy and computational complexity typically grow monotonically until saturation (Page curve).
  • A5 (Horyzont epistemiczny): Stan S0 jest nierestytuowalny ze względu na brak rekordów i bariery złożoności.
  • A5 (Epistemic Horizon): State S0 is irrecoverable due to the absence of records and complexity barriers.
  • A6 (Emergentna lokalność): Graf korelacji indukuje efektywną metrykę; pojęcie „odległości” wyłania się z dynamiki korelacji.
  • A6 (Emergent Locality): The correlation graph induces an effective metric; the notion of “distance” emerges from correlation dynamics.
  • A7 (Zgodność obserwacyjna): Mapowanie zjawisk kosmologicznych na obserwable informacyjne nie narusza znanych ograniczeń (np. z BBN, CMB, LIGO).
  • A7 (Observational Consistency): Mapping cosmological phenomena to informational observables respects known constraints (e.g., from BBN, CMB, LIGO).

A.1 Kod Symulacyjny

A.1 Simulation Code

Poniższy skrypt w języku Python służy do przeprowadzenia symulacji weryfikujących kluczowe aspekty "Modelu Rozsypanych Puzzli".

The following Python script is used to perform simulations verifying key aspects of the "Scattered Puzzle Model".


import numpy as np
import matplotlib.pyplot as plt

def random_unitary_4():
    X = (np.random.randn(4, 4) + 1j * np.random.randn(4, 4)) / np.sqrt(2.0)
    Q, R = np.linalg.qr(X)
    d = np.diag(R)
    ph = d / np.abs(d)
    Q = Q * ph
    return Q

def apply_single_qubit_gate(psi, G, q, N):
    U_parts = [np.eye(2) for _ in range(N)]
    U_parts[q] = G
    U_full = U_parts[0]
    for F in U_parts[1:]:
        U_full = np.kron(U_full, F)
    return U_full @ psi

def apply_two_qubit_gate(psi, gate4, q, N):
    U_parts = []
    idx = 0
    while idx < N:
        if idx == q:
            U_parts.append(gate4)
            idx += 2
        else:
            U_parts.append(np.eye(2))
            idx += 1
    U_full = U_parts[0]
    for F in U_parts[1:]:
        U_full = np.kron(U_full, F)
    return U_full @ psi

def entropy_of_A(psi, k, N):
    M = psi.reshape((2**k, 2**(N - k)))
    U, S, Vh = np.linalg.svd(M, full_matrices=False)
    lambdas = S**2
    lambdas = lambdas[lambdas > 1e-12]
    if lambdas.size == 0:
        return 0.0
    return -np.sum(lambdas * np.log2(lambdas))

def main():
    np.random.seed(42)
    N = 8
    T = 20
    psi = np.zeros(2**N, dtype=complex)
    psi[0] = 1.0
    H = np.array([[1, 1], [1, -1]], dtype=complex) / np.sqrt(2)
    psi = apply_single_qubit_gate(psi, H, 0, N)
    k_values = [1, 2, 4]
    S_A_t = {k: [entropy_of_A(psi, k, N)] for k in k_values}
    for t in range(T):
        i = np.random.randint(0, N - 1)
        U4 = random_unitary_4()
        psi = apply_two_qubit_gate(psi, U4, i, N)
        for k in k_values:
            S = entropy_of_A(psi, k, N)
            S_A_t[k].append(S)
    print("Symulacja zakończona.")

if __name__ == "__main__":
    main()
